🦸‍♂️ Battle of Good and Evil 🦹‍♀️  
*A simple Hero vs. Villain Quiz Game built with HTML, CSS, and JavaScript*

---

## 🎮 About the Game
This is a timed quiz game where the player must decide if a displayed character is a **Hero** or a **Villain**.  
Each round lasts **15 seconds**, and the goal is to correctly classify as many characters as possible before time runs out.  

- ✅ Correct answers increase your score.  
- ❌ Wrong answers do not.  
- ⏱️ The timer counts down — when it reaches 0, the game ends.  
- 🏆 Your **High Score** is saved in local storage between sessions.  

---

## ✨ Features
- Three screens:  
  - **Start Screen** – Begin the quiz.  
  - **Game Screen** – Timer, score, questions, and images.  
  - **End Screen** – Final stats (score, answered, accuracy) + Play Again.  
- **Background music** during gameplay (toggleable).    
- Resettable **High Score** button.  
- Keyboard shortcuts:  
  - `H` or `←` = Hero  
  - `V` or `→` = Villain  
- Mobile-friendly with responsive design.  

---

## 🛠️ Tech Stack
- **HTML5** – structure of the game.  
- **CSS3** – styling with a DC-inspired color theme (blue, red, yellow).  
- **JavaScript (ES6)** – game logic and DOM manipulation.  
- **LocalStorage** – to save and load high scores.
